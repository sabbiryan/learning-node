# learning-node

### install
<ul>
  <li>npm install</li>  
</ul>


### build
<ul>
  <li>npm run compile</li>  
</ul>


### default port
<ul>
  <li>http://localhost:3000</li>
</ul>


### apis
<ul>
  <li>http://localhost:3000/api/users</li>
</ul>


### unit test
<ul>
  <li>.\node_module\.bin\jasmine init</li>
  <li>.\node_module\.bin\jasmine start</li>
  <li>npm test</li>
</ul>
